Your library, reimagined.

LiftOff is a gamepad-first Windows game launcher built for handheld PCs and desktops. Steam, Xbox, Battle.net, and more — unified in one cinematic interface.

⬇ Download for Windows View on GitHub
Why LiftOff

Built for the couch.
Polished for the screen.

Every detail is designed to work with a gamepad in hand — from navigation to artwork to launch animations.

🎮
Gamepad-First Navigation

Full controller support with D-Pad, thumbstick, LB/RB tab switching, and button hints. Designed for ROG Ally, Xbox Ally, and any Windows handheld.

🗂️
Unified Library

Steam, installed Xbox games, Battle.net, and desktop apps all in one place. No more juggling launchers — one interface to rule them all.

🖼️
Cinematic Artwork

Hero artwork, banners, and logos pulled automatically from SteamGridDB. Set custom art for any title in your library.

🎨
Themes & Accents

Sky (light) and Space (dark) themes, each with 5 accent colors and 9 more themes in the pipeline. A launcher that looks as good as your games.

⚡
Instant Launch

Tauri 2 + Rust backend means a featherweight footprint and near-instant startup. No electron bloat, no compromises.

🔓
Open Source

GPL v3 licensed and developed in the open on GitHub. No telemetry, no accounts required, no subscriptions. Just a launcher that works.
